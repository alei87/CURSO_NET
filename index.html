IMPLEMENTACIÓN DE SOLUCIONES DE MOVILIDAD CON TECNOLOGÍAS WEB

OBJETIVOS

El desarrollo web certificado para la W3C para móviles se ha convertido en una alternativa al desarrollo de aplicaciones nativas para los diferentes dispositivos por su simplicidad y ahorro de costes, facilitando la creación de aplicaciones para diferentes dispositivos móviles.

PRERREQUISITOS

Estar familiarizado con el desarrollo de aplicaciones y/o páginas web y las herramientas TIC.
